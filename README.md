# Multi Category Product Scraper

This project is a Python-based web scraper that extracts product data
from multiple categories with pagination support.

## Features
- Category extraction
- Pagination handling
- Data cleaning
- Price conversion
- Rating conversion
- CSV export

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Output
Exports structured CSV file with:
- Category
- Product Title
- Price
- Rating
